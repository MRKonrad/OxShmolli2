Detailed Description

template<typename MeasureType>
class Ox::FunctionsT1< MeasureType >

Container for a model function, cost function and Least-Squares function. And derivatives.

Here model function is defined - calcModelValue(). Fitting algorithms based on optimisation need a cost function - calcCostValue(). Fitting algorithms based on least squares need a residuals calculation - calcLSResiduals(). Some fitting algorithms use derivatives, hence calcLSJacobian() and calcCostDerivative(). The memeber variables are pointers to c-arrays, we need to know how many samples we want to process. Thats the nSamples defined in the constructor.
